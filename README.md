Eine moderne, responsive und druckoptimierte digitale Speisekarte für das mediterrane Restaurant **Daphne** in Bremen.

## 🚀 Live Demo
Die aktuelle Speisekarte ist hier verfügbar:
👉 **[Daphne Speisekarte auf GitHub Pages](https://cschaf.github.io/speisekarte_daphne/)**

## ✨ Features
- **Responsive Design:** Optimiert für mobile Endgeräte, Tablets und Desktops.
- **Print-Ready:** Spezielles Layout für den A4-Druck (PDF-Export).
- **Modernes Design:** Dunkles Farbschema mit Gold-Akzenten und mediterranen Elementen.
- **Auto-Deployment:** Automatisierte Aktualisierung der Webseite via GitHub Actions.

## 🛠 Technologien
- **HTML5 & CSS3** (Vanilla CSS)
- **Google Fonts** (Cinzel, Cormorant Garamond, Lora)
- **GitHub Actions** für CI/CD
